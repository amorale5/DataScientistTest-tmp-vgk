Análisis multivariado — train.csv

Correlación, multicolinealidad, heterocedasticidad, ANOVA/Chi-cuadrado e Information Value, sobre las 45.300 solicitudes de train, orientado a las decisiones de modelamiento de un score de morosidad (default_12m).

1. Correlación entre variables numéricas

Pearson (relación lineal) y Spearman (relación monotónica, más robusta a la fuerte asimetría de varias variables). Se muestra Spearman por defecto ya que es más representativo dado el sesgo de deuda_sistema, ingreso_declarado y monto_solicitado.

Pares con correlación relevante (|r| ≥ 0,35)

2. Correlación de cada variable con el target

Point-biserial (equivalente a Pearson con el target binario) y Spearman. Ordenado por magnitud.

3. Multicolinealidad (VIF)

Variance Inflation Factor sobre el set de variables numéricas (con imputación de mediana solo para este diagnóstico, y log1p en las variables monetarias muy sesgadas). Regla práctica: VIF > 5 es preocupante, VIF > 10 obliga a actuar.

4. Heterocedasticidad

4a. Test de Levene: ¿la varianza de cada variable numérica es distinta entre quienes caen en default y quienes no? 4b. Breusch-Pagan sobre un modelo de probabilidad lineal (OLS de default_12m contra variables clave), para evaluar si un enfoque lineal simple sería apropiado.

4a. Levene (varianza por grupo de default)

4b. Breusch-Pagan (modelo de probabilidad lineal)

5. ANOVA (numéricas por grupo) y Chi-cuadrado (categóricas vs target)

ANOVA de un factor

Chi-cuadrado vs default_12m

6. Information Value (IV) — poder predictivo univariado

Escala estándar de riesgo crediticio: <0,02 sin poder predictivo · 0,02–0,1 débil · 0,1–0,3 medio · 0,3–0,5 fuerte · >0,5 sospechoso de fuga de información (revisar si la variable se conoce realmente antes de otorgar el crédito).
